# aoi_path_demo
* 寻路和视野灯塔demo
* 带有缓冲区域的视野灯塔管理


# unity启动
下载UI包Window-》Package Manager-》选择Unity Registry查找Unity UI选择install安装

# server启动
cd server

npm config set registry https://registry.npmmirror.com

npm install

cd server/src

npm install -g typescript

tsc

node ../dist/app.js
